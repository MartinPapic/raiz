Raíz — Medio Inteligente para Comunicación Sostenible

Impulsado por IA + RAG + Curaduría Humana

Raíz es un medio digital automatizado que combina inteligencia
artificial responsable, tecnologías RAG y curaduría editorial humana
para acelerar la comunicación sobre sostenibilidad, innovación ecológica
y educación ambiental en Chile y Latinoamérica.

Propósito: “Acelerar la comunicación sostenible en Chile y Latinoamérica
mediante inteligencia artificial responsable.”

Público objetivo: - Comunicadores, periodistas y community managers del
área sostenible. - Emprendedores verdes y startups de impacto. -
Fundaciones, instituciones educativas y proyectos ambientales.

Funcionalidades principales: - Generación automática de artículos desde
fuentes verificadas. - IA + RAG para enriquecer artículos con contexto
actualizado y citas reales. - Base vectorial para búsquedas
semánticas. - Panel de curaduría para editar, aprobar y versionar
contenido. - Dashboard de análisis. - Diseño ecológico y accesible.

Stack tecnológico (MVP): Frontend: Next.js, React, TailwindCSS Backend:
FastAPI (Python), Supabase IA/RAG: GPT-4/5, ChromaDB/FAISS Ingesta: RSS,
NewsData.io, Google News API Dashboard: Panel interno en Next.js
Infraestructura: Vercel, Render, Supabase Automatización: Cron jobs

Historias de usuario: Comunicador: recibir artículos sintetizados.
Curador: revisar y aprobar contenido generado. Administrador técnico:
monitoreo del sistema.

Requerimientos funcionales: 1. Gestión de usuarios 2. Ingesta y
generación automática 3. Curaduría humana 4. Publicación 5. Dashboard
analítico 6. Automatización

Requerimientos no funcionales: Rendimiento, escalabilidad,
accesibilidad, sostenibilidad, seguridad, mantenibilidad.

Validaciones MVP: Integración IA-RAG, scraping confiable, pruebas UX,
eficiencia editorial, baja huella digital.

Roadmap: 1. MVP 2. Beta 2.0 3. Escalamiento
